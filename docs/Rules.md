游戏规则


# 游戏目标

回合制游戏, 建造建筑, 生产军团, 攻击敌人, 最终摧毁对方基地.

# 游戏单位

所有单位都有对应的造价(金, 木), 血量和归属方(我方,敌方,中立). 

单位分为建筑和兵种. 

兵种占地为1\*1(但不排除以后会出现占地面积更大的兵种), 拥有血量, 攻击, 攻击范围, 移动范围, 特技 等属性. 大多兵种会有对应的"晋升"形态, 晋升形态的兵种的属性会更高. 数据见 Creatures.csv, 普通和晋升兵种的数据用|符号区分开(没有则代表二者相同), 特技描述中用中括号区分. 标注"@"的特技与攻击冲突(进而与移动也冲突).

建筑的占地面积不尽相同, 建筑通常不能移动, 有些建筑可以攻击, 有些建筑有特殊技能. 数据见 Buildings.csv

# 范围

表示范围的记号主要分为三种, +代表曼哈顿距离, \*代表切比雪夫距离, -代表只能沿直线. 比如\*3为以某一位置为中心的7\*7的正方形范围, 总共49格; 而+2是一个斜45度的一个锯齿方形区域, 总共13格; -5是一个大的十字, 总共21格. 当然也有一些特殊的范围, 比如迈步哥的范围是国象的马步, 不算起始位置总共八格. 当然, 如果这个范围是相对于一个大于1\*1的单位描述的, 那么则会相应的变形. 比如一个2\*2的建筑的+3范围, 事实上(不算建筑本身的四格)有36格.

范围伤害(公主、炸弹塔等)必须全部位于攻击范围内，但溅射伤害(雷龙、导弹井)只需被击打的格子在范围内范围伤害在边界(棋盘或结界的边界)处可以只打一部分，前提是攻击范围允许. 

# 行动规则

每回合可以不限顺序、不限次数地执行以下四种操作，操作可以插入操作：

- 操作兵：
    1. 首先，可以移动这个兵，或者不移动；
	2. 然后，可以让这个兵攻击非己方目标，或者执行特技，或者都不。
	- 每个兵每回合只能操作一次，新获得的兵本回合不能操作。
- 操作建筑:	
    1. 让这个建筑攻击非己方目标
	2. 执行事件或发动特技
	- 每个建筑每回合每项只能操作一次，新生产的建筑本回合不能操作。
- 拆毁一个己方建筑（不返还造价且拒马和城墙不属于己方建筑）
- 发动法术（法术一局只能发动一次） 法术见Spells.csv

# 经典模式

下面介绍经典模式的每回合资源, 初始局面, 卡组构成和晋升规则, 其他模式这些规则可以发生变动. 

## 每回合资源

回合开始时, 获得100(\#金矿+\#总矿)金和5(\#伐木场+\#总矿)木

## 初始局面

1．资源：先手0金0木  后手180金6木. 或者由拍卖同时决定先后手和后手资源数目.

2．摆放：

地图为20\*20方阵.

己方角落布局(方格下标1~20, 笛卡尔坐标系):

- 以(1,1)为左下角的2\*2正方形为基地.
- (1,3),(1,4),(1,5)为金矿.
- (3,1),(4,1),(5,1)为伐木场.
- 以(3,3)为左下角的2*2正方形为兵营.
- (2,5),(5,2)为箭塔.
- (2,3),(3,2)为工兵

对方角落布局与己方中心对称.

在另外两个角落各有一个总矿. 总矿占地2\*2, 离边界有宽度为2的间隙.

游戏开始时总矿为中立建筑，血量30且反伤3（真实伤害）。一方击败中立或敌方矿后获得之，初始血量为10/30，不反伤。// 观星台拆除总矿会使其恢复初始中立状态。

## 卡组

包含8个兵种（其中两个是优势兵种），5个建筑（基地, 金矿, 伐木场, 兵营不用选）和1个法术

选择卡组前双方各Ban 3个兵种或建筑或法术.

## 晋升

1．兵种属性中第二行为晋升兵种数据
2．由兵营产出的兵中，第3n个(骷髅为第3n组)为晋升兵种。若该兵为优势兵种，则改为第2n个(组)。


部分规则解释: 

1. 狂暴比其他改变伤害数值的技能(信标、高B等)优先计算。即先翻倍，再加减
2. 博文诗社不改变目标的血量，不触发鸡男和赏金高B的技能
3. “单位”“目标”指兵和建筑，“友放单位/目标”指我方单位中除了正在描述的单位以外的单位
4. 由操作的定义知，每个工兵每回合最多修一个建筑，且修建筑和攻击是冲突的，且修完建筑后不能移动。朵谢、款同理
5. 骷髅必须在同一个兵营的兵邻处召唤，视为对这个兵营的操作。若空位不足三个则减少召唤数目
6. 范围和溅射伤害击中大型建筑时，只受一次数值最高的伤害
7. 真实伤害不受任何增益和减益影响
8. 闪电塔对后续目标伤害计算不受之前的高B、小土豆等的影响，各种buff改变闪电塔面板伤害(例如狂暴的闪电塔能击中最多5个单位)闪电塔可以导电至本身的范围之外，但不可导电己方单位
